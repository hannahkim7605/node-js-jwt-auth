# JWT Authentication in Node.js
This is a JWT authentication built in Node.js.

## Install pacakages

npm install express
npm install --save-dev nodemon
npm install mongoose
npm install dotenv
npm install joi
npm install bcrypt
npm install jsonwebtoken


## Deployment
git clone
cd node-js-jwt-auth
npm start
